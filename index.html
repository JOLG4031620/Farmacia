Lo básico sobre Sistemas Operativos y Redes de Trabajo

En está página web aprenderas los conceptos e información básica que debes saber sobre los distintos sistemas operativos y sus funciones, así como sus características, ventajas y desventajas, entre otras cosas.

Conceptos básicos

Hardware

Es la parte física de un ordenador o sistema informático. Está formado por los componentes eléctricos, eléctronicos y mecanicos.

Software

Representa toda la parte inmaterial o intangible que hace funcionar a un ordenador, para que realice una serie de tareas específicas.

Sistema Operativo

Gestiona y administra el hardware del dispositivo eléctronico, así como la ejecución de otros programas.

  • software Comercial
  • Software de Libre Distribución
  • Bios
  • Sistemas de Archivos

¿Qué es?

Se conoce como software comercial ó software propietario al tipo de software que tiene como fin una comercialización,es decir, una venta.


Características del software comercial:
  • No permite modificar el código fuente.
  • debes pagar por usarlo, generalmente a través de una licencia.
  • No se puede distribuir sin el permiso del propietario original que lo creó.
  • Ofrece buen nivel de experiencia de usuario y usabilidad.
  • Ofrece respaldo oficial de la empresa que lo desarrollo.

¿Qué es?

Es aquel software donde el usuario del programa tiene la libertad de copiarlo, modificarlo, redistribuirlo y distribuirlo para el beneficio de una comunidad.


Características del software libre:
  • Libertad de ejecutar el programa sin importar el propósito que se tenga.
  • Libertad de acceso al código fuente.
  • Libertad de distribuir el programa, incluso copias de versiones modificadas.
  • Libertad de redistribuir el programa.

¿Qué es?

Es el firmware de la tarjeta madre que funciona por medio del PC y se encarga de almacenar información esencial para que el equipo pueda arrancar correctamente, la BIOS se ejecuta al momento en que se enciende la PC, esta se encarga de revisar de que los perifericos esten operando correctamente.


Funciones de la BIOS:
  • Ver y modifiacr la hora de sistema.
  • Ver la temperatura del procesador.
  • Modificar el voltaje y el consumo del procesador.
  • Modificar le frecuencia de la memoria RAM.
  • Cambiar la configuración de arranque.
  • Otras.

¿Qué es?

Es un método para el almacenamiento y organización de archivos de computadora y los datos que estos contienen, para hacer más fácil la tarea de encontrarlos y acceder a ellos.


Tipos de sistemas de archivos más comunes:
  • Fat32
  • NTFS
  • HFS+
  • EXT2/EXT3/EXT4
  • Brtfs
  • Swap

Redes de Trabajo

Para conocer más sobre las redes de trabajo que existen debemos comprender algunos concetos clave.

Red

Es un sistema de comnicación que se da entre distintos equipos para poder realizar una comunicación eficiente, rapida y precisa, para la transmisión de datos de un ordenador a otro.
